Detailed Description

template<typename Derived>
class cxxutils::singleton< Derived >

The singleton class template.

A class can derive from this class template to become a singleton.

Template Parameters
DerivedThe inheriting class.
See also
GoF: Singleton Pattern
Warning
The singleton pattern is highly criticized and often considered an anti-pattern. So only use it when you are absolutely sure about it and use it with care.
